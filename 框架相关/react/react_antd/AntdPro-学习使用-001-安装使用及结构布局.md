### 安装、启动

```shell
# 全局安装脚手架
$ npm install ant-design-pro-cli -g
# or
$ cnpm install ant-design-pro-cli -g

# 初始化项目
$ pro new

# 安装项目依赖
$ npm install 
# or 
$ cnpm install

# 启动
$ npm start
```
